Drag & Drop With Javascript

Set the draggable attribute to true for the img element and use the ondragstart attribute to call a function from js to allow the drag to occur.


In the js create a function that will prevent the default behavior which is not to allow a drag. Then, create another function that will allow the drag using the .dataTransfer.setData() method. After this, create a function that will get the element by id and use the .dataTransfer.getData() method. Append the drag to the area you wish to drop it to.
